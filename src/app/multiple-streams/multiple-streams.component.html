Multiple Streams

1) Выводить массив последних испускаемых значений их всех 3 потоков когда любой из потоков испускает новое значение.

Начинать выводить значения нужно после того, как все 3 потока испустили хотя бы 1 значение.

2) Вывести последние значения всех 3 потоков когда они завершились.

3) Выводить значения всех 3 потоков когда они все испустили новые значения.
